# Malice (Beta)
A simple interface for [Icestorm/iceprog] written in Vala.

Supports iCE40-HX8K Breakout Board and iCEStick Evaluation Kit, so far.

# Dependencies
Vala 0.36.3 or higher, GTK+ 3.0 or higher. Tested on macOS and elementary OS.

# Usage
Using bash...

```bash
    make
    ./Malice
```
